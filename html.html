HTML Dasar

HTML, atau HyperText Markup Language, adalah bahasa standar yang digunakan untuk membuat dan merancang halaman web. HTML menggunakan tag untuk mengatur struktur konten, seperti teks, gambar, dan tautan, yang kemudian ditampilkan oleh browser web.

Berikut adalah materi dasar HTML yang perlu dipahami:

  1. Struktur Dasar Dokumen HTML:
    • <!DOCTYPE html>: Deklarasi HTML5
    • <html>: Tag pembuka dan penutup
    • <head>: Berisi info dokumen
    • <body>: Berisi konten utama
  2. Tag Dasar HTML:
    • Heading: <h1> sampai <h6>
    • Paragraf: <p>
    • Tautan: <a>
    • Gambar: <img>
    • Daftar: <ol>, <ul>, <li>
    • Tabel: <table>, <tr>, <th>, <td>
    • Divisi: <div>
  3. Tag dan Elemen:

    Tag adalah tanda seperti <p>, sedangkan elemen adalah struktur lengkap seperti <p>Ini paragraf</p>.

  4. Atribut:

    Atribut memberikan informasi tambahan, misalnya src dan alt dalam <img>.

  5. Semantik HTML:

    Gunakan tag semantik seperti <article>, <nav>, dan <aside> sesuai fungsinya.

  6. Pemahaman Dasar:

    HTML bekerja bersama CSS untuk gaya dan JavaScript untuk interaksi.
