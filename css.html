CSS Dasar

CSS (Cascading Style Sheets) digunakan untuk mengatur tampilan dan layout halaman web. Dengan CSS, kita dapat mengubah warna, ukuran, spasi, dan posisi elemen-elemen HTML.

  1. Selector dan Properti: Selector memilih elemen HTML, lalu kita terapkan properti dan nilainya. Contoh: 
    p {
  color: blue;
  font-size: 16px;
}
  2. Penempatan CSS:
    • Inline: di dalam tag HTML, contoh: <p style="color:red;">
    • Internal: dalam tag <style> di <head>
    • Eksternal: menggunakan file .css terpisah
  3. Box Model: Setiap elemen HTML dianggap sebagai kotak: content, padding, border, dan margin.
  4. Layout: Gunakan display, position, flexbox, atau grid untuk mengatur tata letak.
